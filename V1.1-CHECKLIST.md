# Tidal Orbit — v1.1 checklist

v1.0 shipped (iPhone-only, 148 countries, no IAP/ads, Game Center off). These
are the follow-ups, each fairly self-contained. Everything is stubbed/flagged in
code already, so it's mostly "swap the stub / flip the flag" + App Store Connect

## 1. Fix: music inaudible on the phone speaker (plays fine in headphones)
**Cause:** the music tracks lean on deep sub-bass (bass drone + sub-octave sine
around 30–65 Hz). iPhone speakers physically can't reproduce those low
frequencies, so the music vanishes on speaker but is fine on headphones. SFX
survive because they have higher-frequency content.

**Fix (in `tidal/fx.js`, the `TRACKS` table + `musicStep`):**
- Raise the music up an octave (or two) so the core arpeggio sits ~200–800 Hz.
- Keep the bass drone but add a **mid-range harmonic** (e.g., an octave up) so the
  bassline carries on small speakers; don't rely on the ~30–65 Hz sub alone.
- Optionally reduce/remove the `root / 2` sub-octave voice, or move it up.
- Re-test on the actual device speaker, not just headphones/desktop.
